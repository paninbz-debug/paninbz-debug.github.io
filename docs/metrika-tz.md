# ТЗ: настройка Яндекс.Метрики и Директа для сайта LUMO

Для бота/исполнителя с доступом к API Яндекс.Метрики и Яндекс.Директа.

## 1. Создать счётчик Метрики
- **Сайт:** `хромирование-москва.рф` (punycode: `xn----7sbbfcqscungiichud1a5d.xn--p1ai`)
- **Часовой пояс:** Europe/Moscow
- **Опции включить:** Вебвизор, карта кликов, точный показатель отказов, трекинг внешних ссылок.
- На сайте код Метрики уже встроен (`/js/analytics.js`) и активируется автоматически, как только в нём будет указан номер счётчика. **Верни созданный номер счётчика (ID)** — его нужно вписать в `js/analytics.js` в строку `const YM_ID = 0;` (заменить 0 на номер).

## 2. Создать цели (тип «JavaScript-событие», идентификатор = reachGoal)
Сайт уже отправляет эти события через `ym(ID,'reachGoal',<идентификатор>)`. Нужно создать цели с ТОЧНО такими идентификаторами:

| Идентификатор цели       | Что значит                                   | Ценность |
|--------------------------|----------------------------------------------|----------|
| `form_submit`            | Отправлена заявка с формы сайта (главная)     | высокая  |
| `form_submit_from_calc`  | Отправлена заявка из калькулятора             | высокая  |
| `form_submit_failed`     | Сбой отправки заявки (для контроля)           | служебная|
| `phone_click`            | Клик по номеру телефона                       | средняя  |
| `whatsapp_click`         | Клик по кнопке WhatsApp                       | высокая  |
| `telegram_click`         | Клик по кнопке Telegram                       | высокая  |
| `calc_to_form`           | Открыл форму из калькулятора                  | средняя  |
| `portfolio_filter`       | Использовал фильтр портфолио                  | низкая   |
| `portfolio_view`         | Кликнул по работе в портфолио                 | низкая   |
| `scroll_50`              | Доскроллил до 50% страницы                    | низкая   |
| `scroll_90`              | Доскроллил до 90% страницы                    | низкая   |

Главные конверсии для оптимизации рекламы: **form_submit, form_submit_from_calc, whatsapp_click, telegram_click, phone_click.**

## 3. Яндекс.Директ
- Связать аккаунт Директа со счётчиком Метрики.
- В кампаниях задать **ключевые цели** = перечисленные «высокие» конверсии (form_submit, *_from_calc, whatsapp_click, telegram_click, phone_click).
- Включить автостратегию с оптимизацией по этим конверсиям.
- Настроить UTM-разметку объявлений (сайт уже корректно обрабатывает utm_* — в robots.txt заданы Clean-param для Яндекса).

## 4. После настройки
1. Вписать номер счётчика в `js/analytics.js` (`YM_ID`).
2. Проверить в отчёте Метрики «Конверсии», что цели фиксируются (отправить тестовую заявку и кликнуть мессенджеры).
