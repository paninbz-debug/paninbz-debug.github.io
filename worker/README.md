# LUMO — приём заявок в Telegram через Cloudflare Worker

Форма на сайте отправляет заявку не напрямую в Telegram, а на **Cloudflare Worker**.
Токен бота и chat_id лежат секретами на стороне Cloudflare и **не видны на сайте**.

## Что понадобится
- Бот: **@Lumochrome_bot** (токен у тебя есть).
- `chat_id` для заявок: **687941614** (личка Данила, @Panin_off).
  Чтобы слать в группу — добавь бота в группу и подставь её chat_id (начинается с `-100...`).

## Шаги (≈5 минут)

1. Зайди на **dash.cloudflare.com** → раздел **Workers & Pages** → **Create** → **Create Worker**.
2. Назови, например, `lumo-lead`. Нажми **Deploy** (создастся заготовка).
3. Открой **Edit code**, удали весь шаблон и вставь содержимое файла [`lead-bot.js`](./lead-bot.js). Нажми **Deploy**.
4. Открой **Settings → Variables and Secrets** воркера и добавь два **Secret**:
   - `BOT_TOKEN` = токен от @BotFather
   - `CHAT_ID`   = `687941614`
   - (необязательно) `ALLOW_ORIGIN` = `https://xn----7sbbfcqscungiichud1a5d.xn--p1ai`
   После добавления секретов нажми **Deploy** ещё раз.
5. Скопируй URL воркера — он вида `https://lumo-lead.<твой-сабдомен>.workers.dev`.
6. Пришли мне этот URL — я пропишу его в форму (одна строка в `js/main.js`).
   Либо сам впиши его в `js/main.js` в строку `LEAD_ENDPOINT`.

## Проверка
После прописанного URL отправь тестовую заявку с сайта — сообщение должно прийти в Telegram.
Проверить воркер можно и вручную:

```bash
curl -X POST 'https://lumo-lead.<сабдомен>.workers.dev' \
  -H 'Content-Type: application/json' \
  -d '{"name":"Тест","phone":"+79990000000","service":"Проверка"}'
# Ожидаемо: {"ok":true}
```

## Безопасность
- Токен и chat_id **не хранятся в репозитории и не отдаются браузеру**.
- Скрытое поле-ловушка (`company`) отсекает часть спам-ботов.
- Лимиты длины полей — на стороне воркера.
- Рекомендуется задать `ALLOW_ORIGIN` (тогда воркер примет запросы только с твоего домена).
